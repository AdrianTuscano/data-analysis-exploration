# data-analysis-exploration

In this repository I explored the basics of matplotlib to visualize data, as well as other tools like sklearn decision trees, pandas dataframes, and numpy. 

Initially I looked into how the matplotlib library worked. 

Then I used some NBA data and created my own NBA statistic with abritrary constants to visualize the most and least ethical players in the association. 

Lastly I used sensor reading data from (UC Irvine)[https://archive.ics.uci.edu/dataset/194/wall+following+robot+navigation+data__] (trying my best to not look at the title, additional info, or any other hints) and tried to draw my own conclusions on what the robot was doing.
